◆ RecruitAgents prototype

AI-agents voor je recruitmentproces

Van gesprek naar ATS-klare output.
In seconden.

Drie gespecialiseerde agents die het handwerk uit recruitment halen: screening, intakeverslagen en kandidaatcommunicatie. Gebouwd als prototype voor integratie met je eigen ATS.

PDF of Word (.doc, .docx)
De matchanalyse verschijnt hier: score, sterke punten, aandachtspunten en gerichte screeningsvragen.
Het gestructureerde verslag verschijnt hier: samenvatting, kandidaatprofiel, actiepunten en een ATS-klare notitie.
Hier verschijnen een e-mail- en LinkedIn-variant van je outreachbericht.

ATS-integratie

Dit prototype draait volledig client-side. Voor productie koppel je de agents via een backend of automatiseringslaag (bijv. n8n) aan je ATS: webhook in, gestructureerde JSON uit, kandidaatdossier bijgewerkt. Elke agent levert output als gestructureerde data — klaar om te mappen op je ATS-velden.

RecruitAgents · intern prototype · aangedreven door de Anthropic API

Instellingen

⚠ Je API-key wordt alleen in deze browsersessie bewaard (sessionStorage) en gaat rechtstreeks naar api.anthropic.com. Gebruik dit alleen voor demo-doeleinden — zet voor productie een backend-proxy (bijv. n8n) ertussen.
